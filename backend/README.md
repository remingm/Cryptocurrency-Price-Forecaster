# stonk-pix-ai

this is the repo for the revolutionary stonk pixer. utilizing ai, blockchain, cloud, and synergy

to install:

`npm install`

to run:

`npm run start:dev`

# docker usage

to build:

`docker build . -t stonk-pix-ai-backend`

to start:

`docker run -p 3000:3000 -d stonk-pix-ai-backend`

then visit localhost:3000/
